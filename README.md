# Базы данных — учебный проект

## Описание проекта

Учебный проект по дисциплине «Базы данных». В репозитории собраны SQL-скрипты для создания таблиц, наполнения их тестовыми данными и выполнения аналитических запросов. Каждая задача оформлена в отдельном файле и содержит варианты DDL для **MySQL** и **PostgreSQL**.

## Цели

- Закрепить навыки проектирования реляционных схем данных.
- Научиться писать запросы разной сложности: фильтрация, агрегация, объединение таблиц, рекурсивные CTE.
- Отработать полный цикл работы с БД: создание структуры → заполнение → выборка.
